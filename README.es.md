# Code Symphonist

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Code Symphonist para OpenClaw.

## Qué resuelve
Orquestra la refactorización de múltiples archivos mientras preserva la integridad arquitectónica.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque centrado en la seguridad
- Soporte para rollback

## Uso
/code-symphonist

## Ejemplo
```
Analiza el código para identificar problemas de programación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
